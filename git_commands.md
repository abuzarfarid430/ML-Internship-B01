\# Essential Git Commands



1\. \*\*git init\*\*  

&nbsp;  Initializes a new Git repository.



2\. \*\*git clone <url>\*\*  

&nbsp;  Copies a remote repository to your local machine.



3\. \*\*git status\*\*  

&nbsp;  Shows the current status of files.



4\. \*\*git add .\*\*  

&nbsp;  Stages all changes for commit.



5\. \*\*git commit -m "message"\*\*  

&nbsp;  Saves changes with a message.



6\. \*\*git branch\*\*  

&nbsp;  Lists all branches.



7\. \*\*git checkout <branch-name>\*\*  

&nbsp;  Switches to another branch.



8\. \*\*git pull\*\*  

&nbsp;  Fetches and merges changes from GitHub.



9\. \*\*git push\*\*  

&nbsp;  Uploads local commits to GitHub.



10\. \*\*git merge <branch>\*\*  

&nbsp;   Merges another branch into the current branch.
